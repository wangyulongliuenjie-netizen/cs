# Personal Website

This project is a personal website that showcases my portfolio and provides information about my skills and experiences.

## Project Structure

- `index.html`: The main page of the personal website, containing the basic structure and content such as the title, navigation bar, and main content area.
- `css/styles.css`: This file contains the style definitions for the website, including layout, colors, fonts, and other visual effects.
- `js/script.js`: This file includes the JavaScript code for interactive features, such as button click events and form validation.

## Usage

To view the website, open the `index.html` file in a web browser. You can modify the content in `index.html`, adjust styles in `css/styles.css`, and add interactivity through `js/script.js`.
